# FORM·RIFT

**Geometric Roguelite — 10 Stages**

A browser-based roguelite where you draw triangles to attack enemies.

## How to Play
- **WASD** — Move
- **Mouse** — Aim
- **Left Click × 3** — Draw a triangle attack zone
- **Right Click** — Shield / Ultimate (when gauge full)
- **Hold E** — Charge spell (after unlocking)

## Features
- 10 stages with increasing difficulty
- Elite boss (Stage 5) + Final boss (Stage 10)
- Blessing system — choose upgrades between stages
- Spell system — trace patterns to cast Time Rewind / Invincible
- Real-time delta timing for consistent gameplay speed
